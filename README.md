# T1 Frontend

### 1. Сборка приложения фронтенда 
```shell
docker build -t t1-frontend .
```

### 2. Запуск Docker compose
```shell
docker run -d -p 8080:80 t1-frontend
```

### 3. Приложение доступно по адресу
```
http://localhost:8080/
```
